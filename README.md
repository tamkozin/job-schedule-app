# job-schedule-app

## 目的

就活のタスク（応募・面接・締切）を管理し、進捗を可視化できる。


## 機能　(MVP段階)

1. ユーザ登録・ログイン
2. タスク登録 （会社名・ステータス・期限・メモ）
3. タスク一覧表示
4. タスク削除
5. ログアウト
